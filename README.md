### Installation
```bash
# Install for every installed php version >=8.2
curl -sSfL https://github.com/rootfebri/phprfs-ext/raw/refs/heads/main/install | sudo bash
```
```bash
# Install for current installed php version (at least >=8.2)
curl -sSfL https://github.com/rootfebri/phprfs-ext/raw/refs/heads/main/install | sudo bash -s -- current
```
